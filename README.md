## 🛠️ **Instalación Rápida**

```bash
# 1️⃣ Clona el repositorio
git clone https://github.com/JosiasDenis12/TuuxKabin.git

# 2️⃣ Abre en Android Studio
cd TuuxKabin

# 3️⃣ Configura tu Google Maps API Key
# Agrega tu clave en local.properties:
MAPS_API_KEY=tu_api_key_aqui

# 4️⃣ Ejecuta la aplicación
# Presiona Run en Android Studio o usa:
./gradlew installDebug
```

## 🏗️ **Arquitectura de la Aplicación**

```
📱 TuuxKabin App
├── 🎨 UI Layer (Activities + Fragments)
│   ├── MainActivity (Dashboard)
│   ├── ExplorarCenotesActivity (Exploration)
│   ├── ServiciosActivity (Services)
│   ├── MapsActivity (Google Maps)
│   └── LoginActivity (Authentication)
├── 🔧 Business Layer (Logic)
│   ├── Adapters (RecyclerView)
│   ├── Models (Data Classes)
│   └── Utils (Helpers)
└── 💾 Data Layer
    ├── Local Database (SQLite)
    ├── SharedPreferences
    └── Google Maps API
```
